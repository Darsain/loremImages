LoremImages

jQuery plugin for loading random images from LoremPixel service

Random images from all categories (simplest call without any options)

$("#random").loremImages( 94, 150 );

B&W images from nightlife category using custom item template

$("#nightlife").loremImages( 260, 180, {
	count:       8,
	grey:        1,
	category:    'nightlife',
	itemBuilder: function(i,url){
		return '<li class="span3"><a href="http://lorempixel.com" class="thumbnail"><img src="'+url+'"></a></li>';
	}
});
