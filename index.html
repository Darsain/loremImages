LoremImages

jQuery plugin for loading random images from LoremPixel service

Random images from all categories

$("#random").loremImages( 94, 150 );

Nightlife images with random width value

$("#nightlife").loremImages( 100, 150, {
	count:        8,
	randomWidth:  200,
	category:     'nightlife'
});
